## 补丁目的
对《猫头鹰男孩》的官方中文进行全面重译。

## 安装方法
1. 下载【Owlboy】文件夹内的文件，覆盖原来的游戏文件。

## 补充说明
* 补丁对应操作系统为Windows，对应游戏平台为Steam。
* 补丁为完全汉化，支持Steam成就解锁。
* 补丁对应的游戏内容版本ID可在【游戏库 > 右键属性 > 本地文件】的左下方查看。
